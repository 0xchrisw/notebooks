NAME
       plutil - A converter tool for binary or XML Apple property lists

SYNOPSIS
       plutil -i|--infile in_file.plist -o|--outfile out_file.plist [--debug]

OPTION
       -i or --infile
	      The file to read in.

       -o or --outfile
	      The file to convert to.

       -d, -v or --debug
	      Provide extended debug information.

DESCRIPTION
       plutil
