NAME
       blueman-assistant - application for configuring and pairing bluetooth
       devices

SYNOPSIS
       blueman-assistant [options]

DESCRIPTION
       blueman-assistant configures specified device, if there is no device
       specified, it will display a dialog to choose from.

OPTIONS
       --device=ADDRESS
	      Address of the device to configure
