NAME
       zsysctl - ZFS SYStem integration control zsys daemon

SYNOPSIS
       zsysctl COMMAND [flags]

DESCRIPTION
       Zfs SYStem tool for an enhanced ZFS on Linux experience.
	It allows running multiple ZFS system in parallels on the same
       machine,
	get automated snapshots, managing complex zfs dataset layouts
       separating
	user data from system and persistent data, and more.

OPTIONS
       -h, --help[=false]
	   help for zsysctl

       -v, --verbose[=0]
	   issue INFO (-v) and DEBUG (-vv) output
